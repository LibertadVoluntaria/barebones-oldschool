barebones-oldschool
================

An OldSchool RuneScape client that is simply a client.

### Usage
```
$ java -jar OldSchool-RuneScape.jar [world number]
```

### Dependencies
Java 1.8.0+.

### Build
Build using Ant.
```
$ ant build
```
